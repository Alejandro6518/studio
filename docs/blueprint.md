# **App Name**: Ruta Sin Miedo

## Core Features:

- Página de Inicio Interesante: Una interfaz visualmente atractiva que introduce la pregunta central del proyecto y su justificación, invitando al usuario a iniciar la exploración.
- Explorador de Dilemas (IA): Una herramienta de IA que genera breves dilemas personales o éticos adaptados para el usuario, que invitan a tomar una decisión hipotética.
- Asistente de Reflexión (IA): Tras interactuar con un dilema, una herramienta de IA ofrece preguntas abiertas y comentarios personalizados basados en la elección del usuario para fomentar la autorreflexión sobre el miedo al error.
- Perspectivas & Consejos: Una sección que muestra citas inspiradoras, resúmenes de estrategias para superar el miedo a equivocarse y recursos que invitan a una mentalidad de crecimiento.
- Experiencias Compartidas (Curadas): Presentación de anécdotas o testimonios anónimos y curados (predefinidos para MVP) relacionados con el miedo a tomar decisiones, creando un sentido de comunidad y empatía.

## Style Guidelines:

- Esquema de color claro para evocar claridad y superación. Color principal: un azul sereno que invita a la reflexión (#3973A6). Color de fondo: un azul muy claro y desaturado para mantener la ligereza (#EEF3F7). Color de acento: un aqua brillante y distinto para captar la atención en elementos interactivos (#45C5E2).
- Fuente para títulos y encabezados: 'Alegreya' (serif) para una sensación elegante e intelectual. Fuente para el cuerpo del texto y las reflexiones: 'Inter' (sans-serif) para asegurar una legibilidad moderna y objetiva.
- Iconos limpios y minimalistas que simbolicen conceptos como elección, camino, reflexión, duda y resolución, para mantener la atención en el tema central sin recargar la interfaz.
- Diseño espacioso y responsivo con amplio uso de márgenes y paddings para un respiro visual. El contenido clave se presenta en tarjetas o bloques bien definidos, con enfoque en un flujo de información guiado y lineal para el usuario.
- Transiciones sutiles y elegantes al navegar entre secciones o al interactuar con los dilemas. Pequeñas animaciones al revelar prompts de reflexión o al confirmar una elección para realzar la experiencia de usuario sin distraer.
